#SmartATM System (Java) DecodesLab_Project_3
#Project Overview
SmartATM is a console-based Java application that simulates real-world ATM functionalities. It allows users to perform secure banking operations like withdrawals, deposits, and balance checks.

#Features

* User authentication (PIN-based login)
* Balance inquiry
* Cash withdrawal
* Deposit functionality
* Transaction validation
* Simple and interactive CLI interface

#Technologies Used

* Java (Core Java)
* OOP Concepts

#How to Run

1. Open terminal / CMD
2. Compile:

   ```
   javac SmartATM.java
   ```
3. Run:

   ```
   java SmartATM
   ```

#Project Structure

```
SmartATM.java
```

#Future Enhancements

* GUI interface (Java Swing / JavaFX)
* Database integration (MySQL)
* Multi-user support
