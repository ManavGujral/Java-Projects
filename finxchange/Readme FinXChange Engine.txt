# FinXChange Engine

FinXChange Engine is a simple currency converter application built using Java. It is designed using object-oriented programming concepts and provides accurate currency conversion using BigDecimal for precision.

---

## Features

- Convert between multiple currencies (USD, INR, EUR, GBP)
- Uses BigDecimal for accurate financial calculations
- Clean and modular code structure
- Console-based user interface
- Basic error handling and input validation
